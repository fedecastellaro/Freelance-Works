## FULL ESP32 BOARD

The PCB includes:

- SIM800C.
- SPI Headers.
- SD card.
- GPIO I2C Multiplexer.
- USB-UART converter.
- Fully integrated battery charger module.
- Different sources of power alimentation (220V AC, 5VDC, USB 5V)
- 3 general purpose push buttons
- 3 general purpose leds
